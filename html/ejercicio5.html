Ejercicio 5: Estructura compleja y semántica

Objetivo: Construir el esqueleto de una página web compleja utilizando solamente HTML semántico (sin CSS), que simule un diseño de tres columnas.
  • Crea un nuevo archivo llamado "ejercicios5.html
  • Dentro del <body>, utiliza etiquetas semánticas para definir las secciones principales de una página: header, nav, main, article, section, aside y footer.
  • El header debe contener un encabezado <h1> y un subtítulo <p>
  • Dentro de la etiqueta <main>, crea una estructura que simule un diseño de tres columnas. Esto se puede lograr anidando etiquetas como <div> o incluso usando una tabla para la maquetación.
    • La columna de la izquierda podría ser un <article>.
    • La columna del centro podría ser una <section>.
    • La columna de la derecha podría ser un <aside>.
  • Dentro de cada una de estas columnas (<article>, <section>, <aside>), añade encabezados, párrafos y listas.
  • Añade un <footer> con información de derechos de autor y algún enlace adicional.
