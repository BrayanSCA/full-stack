Ejercicio 2: Listas y Enlaces

Objetivo: Combinar texto con listas y enlaces para una navegación simple.
  • Crea un nuevo archivo llamado "ejercicio2.html"
  • Utiliza un encabezado de nivel 2 para un título que diga "Mis enlaces favoritos"
  • Crea una lista desordenada de 3 items
  • Cada ítem de la lista debe contener un enlace a una página web diferente (por ejemplo, Google, Wikipedia y tu sitio web favorito). El teto del enlace debe ser el nombre del sitio web.

Mis Enlaces Favoritos

  • GitHub
  • Web.dev cursos programación
  • FreeCodeCamp curso inglés
