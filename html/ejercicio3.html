Ejercicio 3: Tablas

Objetivo: Estructurar datos de forma tabular.
  • Crea un archivo llamado "ejercicio3.html".
  • Crea una tabla para mostrar un horario semanal.
  • la tabla debe contener 3 columnas: "dia", "actividad" y "hora". Utiliza etiquetas de encabezado <th> para los títulos de las columnas.
  • añade por lo menos 3 flas de datos <tr> para la tabla. Cada fila debe contener las celdas de datos <td> correspondientes al día, actividad y hora.
  • Por ejemplo: una fila podría ser "Lunes", "Estudiar HTML", "10:00 AM"
