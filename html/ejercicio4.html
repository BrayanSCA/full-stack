Ejercicio 4: Formularios

Objetivo: Recopilar información del usuario usando un formulario simple.
  • Crea un nuevo archivo llamado ejercicio4.html
  • Dentro del <body>, crea un formulario
  • Añade una etiqueta <label> y un campo de entrada de texto <input> para que el usuario ingrese su nombre.
  • Añade una etiqueta y un campo de entrada de email.
  • Incluye un campo de contraseña
  • Crea un botón de tipo <submit> para enviar el formulario.
